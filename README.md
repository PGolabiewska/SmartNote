# SmartNote  <img width="45" height="45" alt="note-img" src="https://github.com/user-attachments/assets/d990a7a1-b10a-402e-8e53-35e98a0eea11" />



SmartNote is a clean and intuitive web application built with React. 

It allows users to manage daily notes, write down idea and create to-do lists within a modern, responsive interface.

> [!NOTE]
> This project is still under development, you can find the list of functionalities below.

<br/>

## Roadmap
- [x] Environment Setup: Configured Vite + React.
- [x] Component Architecture: Divided UI into Header, Footer, and App components.
- [x] Styling (CSS): Custom styles & fonts.
- [x] Static UI: Rendering placeholder data to visualize the layout.

<br />

- [ ] Add Notes: Create a component to handle input state and submission.
- [ ] Delete Notes: Implement a filtering function to remove notes (by unique ID?).
- [ ] Edit Functionality: Allow users to modify the content of existing notes.
- [ ] Validation: Prevent users from submitting empty notes.
- [ ] Data Persistence: Integrate LocalStorage to save notes between browser sessions.
- [ ] Light Mode / Dark Mode: Implement a theme switcher for better user experience.

<br />

## Technologies Used
- `React 19`
- `JavaScript (ES6+)`
- `Vite`
- `CSS3`
- `HTML5`
